## Project setup :wrench:
```
# Clone the repo
> git clone https://github.com/CodexUdeA/codexudea.github.io.git

# Install dependencies
> npm install

# Compiles and hot-reloads for development
> npm run serve
 
# Compiles and minifies for production
> npm run build

# Lints and fixes files
> npm run lint
```
